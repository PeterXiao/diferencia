= Install
include::_attributes.adoc[]

To install Diferencia you only need to go to release page onhttps://github.com/lordofthejars/diferencia/releases[GitHub Release] select the platform and the version and download it.

Then you just need to rename the file to something more meanful name such as `diferencia` and add it to your `PATH` directory.

== Linux and Mac OS X
In case of `Linux` and `Mac OS X` you can run next commands:

[source, bash]
----
mv diferencia_XXX_amd64 diferencia
chmod 755 diferencia
mv diferencia /usr/local/bin/diferencia
----

== Windows
In case of `Windows` you can run next commands:

[source, bash]
----
ren diferencia_windows_amd64.exe diferencia.exe
set PATH=%PATH%;<diferencia_directory>
----

[#docker]
== Docker

Diferencia is also provided as Docker image.

For example:

`~ $  docker run -ti --rm -p 8000:8000 lordofthejars/diferencia:<version>` 

Just start Test Hub server at port `8080`.
